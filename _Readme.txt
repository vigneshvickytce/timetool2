As of now, there is no setup program.
Simply copy the program to its own directory 
and make a shortcut to TimeTool.jar on your desktop,  
quicklaunch bar, an/or Start menu.

From the command line, cd to the TimeTool directory
and run:
 java -cp . TimeTool.jar
